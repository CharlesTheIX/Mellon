MELLON - QUICK HELP

USAGE
mellon [command] [args]
mellon # REPL mode

BUILT-IN COMMANDS
help                 Show this help
exit | :q            Exit
repl Enter           REPL
config               Edit config in editor
config set k=v [...] Set config keys
config source        Reload config
file-system | fs     File operations

FS COMMANDS
fs read --path=FILE
fs write --path=FILE [--editor=vim|nvim|code]
fs copy --from=SRC --to=DEST
fs delete --path=FILE
fs get_abs --path=PATH

CONFIG (~/.mellonrc)
editor=vim|nvim|code
prompt=⚡
show_intro=true|false

TIPS
Unrecognized commands are passed to the shell.
Use arrow keys to navigate REPL history.
