MELLON HELP

USAGE
	mellon [command] [args]
	mellon            # REPL mode

BUILT-IN
	benchmark | bench    Time a command
	help                 Show this help
	exit | :q            Exit
	repl                 Enter REPL
	naselaska           Launch NaseLaska GUI
	config               Edit config in editor
	config set k=v [...] Set config keys
	config source        Reload config
	file-system | fs     File ops

FS
	fs read    --path=FILE
	fs write   --path=FILE [--editor=vim|nvim|code]
	fs copy    --from=SRC --to=DEST
	fs delete  --path=FILE
	fs get_abs --path=PATH

CONFIG (~/.mellonrc)
	editor=vim|nvim|code
	prompt=⚡
	show_intro=true|false
	show_cwd=true|false

TIPS
	Unrecognized commands go to the shell.
	Benchmark prints elapsed time.
	Arrow keys navigate history.

NASELASKA
	Experimental GUI mode built with raylib.
	Run: mellon naselaska
	Opens 800x600 window for game/GUI interaction.
