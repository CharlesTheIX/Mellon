MELLON

Interactive Zig shell with built-in config, file helpers, search, and shell fallback.

Quick start:
	help            Show the command guide
	fs help         Show file-system commands
	search prompt   Search the current tree with ripgrep
	config          Open ~/.mellonrc in your editor
	exit            Quit Mellon

Tips:
	Unrecognized commands are passed to your shell.
	Arrow keys navigate command history.
	Ctrl+C clears the current input line.
