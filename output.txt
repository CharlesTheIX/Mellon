[0m[37m# Welcome!

Interactive Zig shell with built-in config, file system utilities, and more.

## Quick start:

- help Open command guide
- exit | :q Quit REPL mode
- config Open ~/.mellonrc in your editor

## Tips:

- Unrecognized commands are passed to your shell.
- Ctrl+C kills the REPL process and all child processes.
- Arrow keys navigate command history (if set in your config file).
